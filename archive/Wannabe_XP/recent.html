Almost everything you find here is hand crafted by me using vanilla HTML, CSS, and JavaScript. The only exceptions to this statement are for the following resources,

  • Paint: The paint app is actually a third party embeddable widget, which anyone can embed on their website, https://jspaint.app/
  • Internet Explored: The browser uses Old Google (oldgoogle.neocities.org) as it's default search engine
  • Webamp: The webamp app is a tool created by https://webamp.org/.
  • Images & Icons: The following icons are property of Microsoft. Minimize Window, Maximize Window, Close Window, Log off, Shutdown, Enter/Go, Back/Forward, The Windows Logo, and The Wallpaper
  • Chattable, Amdroid, and Neocraft: Are all projects of mine that have been embedded using iFrames.

That's it! Everything else is typed letter by letter in vanilla HTML, CSS, and JS.

About Winbows

About

Welcome to Winbows XP (professional). It's a fun web experience (hosted on Neocities! :D) that brings back that nostalgic feel of technology from the early 2000s. This project is continuously growing. Developed & maintaned by Andrew, this web application is written entirely in client-side JavaScript, HTML and CSS. No, this website does not use XP.css or any other third party source code. Check the credits app for some exceptions. This is not supposed to be a copy of Windows XP, nor an emulator, but rather a fun gimmick with heavy ispiration from the classic Windows XP desktop environment. I do give Microsoft credit for several resources used to create this such as, system icons and the Windows logo. but everything else has been designed by me using CSS. Snoop around, but most importantly have fun!

Note: some windows have been minimized to keep things neat & clean, feel free to explore the opened tabs!

Having Issues?

Try CTRL+SHIFT+R on most browsers (or CTRL+F5 on Firefox) to hard reload. Or, clear your browser's cache.
