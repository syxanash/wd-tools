Almost everything you find here is hand crafted by me using vanilla HTML, CSS, and JavaScript. The only exceptions to this statement are for the following resources,

  • Paint: The paint app is actually a third party embeddable widget, which anyone can embed on their website, https://jspaint.app/
  • Internet Explored: The browser uses Old Google (oldgoogle.neocities.org) as it's default search engine
  • Webamp: The webamp app is a tool created by https://webamp.org/.
  • Images & Icons: The following icons are property of Microsoft. Minimize Window, Maximize Window, Close Window, Log off, Shutdown, Enter/Go, Back/Forward, The Windows Logo, and The Wallpaper
  • Chattable, Amdroid, and Neocraft: Are all projects of mine that have been embedded using iFrames.

That's it! Everything else is typed letter by letter in vanilla HTML, CSS, and JS.
