my primary set of pronouns is a/am/air/airs/amself. if you haven't heard of them before, that's because I made them up myself, based on some old "regional nominative pronoun" (they only found the first one, a, likely the others never existed) that was found in a 14th century text. I thought it would be funny to have a set that was mostly made up of other words, so it wouldn't be spellchecked out but would completely fuck with grammar checks. My other set of pronouns, *e/h*/h*s/h*s/h*self, really only work when used in text-based communication. I found them online, they were made as a half-joke for some 90s chat software. they're also called splat pronouns, a name much easier to look up. feel free to replace the asterisks with whatever other unicode symbols work for the platform/you think are cool, although try to avoid emoji or anything else that would be rendered in color. a few that I think would be cool include ꙮ, ‽, or ⌂. get creative!
